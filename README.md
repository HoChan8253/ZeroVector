# ZeroVector

> 낮에는 파밍과 정비, 밤에는 생존 전투를 반복하며 웨이브를 돌파하는 1인칭 FPS 생존 슈팅 게임입니다.  
> 황폐한 사막 필드에서 적을 처치해 골드를 획득하고, 상점을 통해 무기와 스탯을 강화하며 다음 밤을 준비합니다.  
> 밤이 시작되면 안전 구역이 생성되고 시간이 지날수록 점점 좁혀집니다.
> 플레이어는 제한된 공간 안에서 몰려오는 적과 전투를 이어가야 합니다.  
> 웨이브가 진행될수록 다양한 적이 등장하고, 최종 10웨이브에서는 보스전이 진행됩니다.

## How To Play

1. 낮 시간 동안 필드를 탐색하며 적을 처치하고 골드를 획득합니다.
2. 획득한 골드로 상점에서 무기를 언락하거나 업그레이드합니다.
3. 밤이 되면 안전 구역이 생성되며 전투 페이즈가 시작됩니다.
4. 좁혀지는 구역 안에서 적을 상대하며 생존합니다.
5. 밤이 끝난 뒤 남은 적을 모두 정리하면 다음 웨이브로 넘어갑니다.
6. 10웨이브 보스를 처치하고 남은 적까지 모두 제거하면 게임을 클리어합니다.
